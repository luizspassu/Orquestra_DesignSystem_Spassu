Violações a Corrigir

Problemas identificados no protótipo. Ordenados por severidade.

🔴 CRÍTICO — Cor de texto: #FA4616 sobre fundo branco
Contraste ~3.0:1 — reprovado por WCAG AA (mínimo 4.5:1). Ação: substituir por C.orangeText (#C2410C) em todos os textos de KPI e destaque.
// ❌ textColor: C.orange → #FA4616 — contraste 3.0:1 // ✅ textColor: C.orangeText → #C2410C — contraste 4.5:1
🔴 CRÍTICO — Touch target de ícones: 26×26px (mínimo: 36px)
Viola WCAG 2.5.5. Ação: minWidth/minHeight: 36px em todos os botões de ícone.
// ❌ style={{ minWidth:26, minHeight:26, padding:4 }} // ✅ style={{ minWidth:36, minHeight:36, padding:8 }}
🟡 MÉDIO — Token C.lime não renomeado no JSX
O objeto C ainda usa lime. Renomear para avocado para consistência com este documento e adicionar avocadoDark e orangeText.
// ❌ lime: "#DBE442" // ✅ avocado: "#DBE442", avocadoDark: "#b8c018", orangeText: "#C2410C"
🟡 MÉDIO — orquestra_preview.html desatualizado
HTML de preview não inclui Fase C. O protótipo real está em ContractDetailPage_preview.jsx. Ação: regenerar o HTML.
